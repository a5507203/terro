## Session Information
Session name: terro-website
User Request: Build a multi-page static website for Terro AI using plain HTML/CSS/JavaScript with bold colorful visual style
Plan Version: v1

## Design Doc
Dark theme with electric cyan (#00D4FF), vibrant purple (#8B5CF6), and hot pink (#FF3D71) accents. Glassmorphism cards, gradient text headlines, scroll animations. 5 pages: Home, About, Technology, Use-Cases, Contact. Mobile-first responsive design.

## Technology Stack
HTML5, CSS3 (custom properties, flexbox, grid), Vanilla JavaScript ES6
